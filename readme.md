# Maintenance Page

## Description
This app serves as a global maintenance page catcher.

## Installation
To install the app, use the following command:
```
npm install
```

## Usage
To start the app, run:
```
npm start
```

## Dependencies
- Express: ^4.18.3
- Path: ^0.12.7
